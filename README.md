# 🎬 Movie Hub

A simple Full Stack Movie Review application built as part of my Cognifyz Technologies Internship (Task 5).

## 🚀 Features

- Search Movies
- Movie Reviews
- Responsive UI
- FastAPI Backend
- Fetch API Integration
- Movie Not Found Message
- Enter Key Search

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- FastAPI

## 📂 Project Structure

```
Movie-Hub/
│── index.html
│── style.css
│── script.js
│── main.py
│── images/
└── README.md
```

## ▶️ Run Locally

### 1. Clone Repository

```bash
git clone https://github.com/sarveshwarhirve17-ux/Movie-Hub.git
```

### 2. Install FastAPI

```bash
pip install fastapi
pip install "uvicorn[standard]"
```

### 3. Run Backend

```bash
uvicorn main:app --reload
```

### 4. Open Frontend

Run `index.html` using Live Server.
